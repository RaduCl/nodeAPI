Steps to run the app
=====================
* After cloning the repo, install the dependencies by running **npm install**
* To start the server, run **npm start** on the base directory
* API responds to REST-fullly on /api/books/
* Ex: use /api/books/?genre=SciFi to filter by genre, author or title

Perquisites
============
The server assumes that you have a local mongo instance running. This means if you have mongo installed locally and the mongod daemon running
